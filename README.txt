Objectives:
Add a user-friendly GUI to the pre-existing code from the API
to provide a better quality-of-life experience than the output box of the code

Additions:
-Adds a graphical interface for the user upon running the code
-Allows the user to input multiple stops in the event they would like to stop at another location
before arriving at their destination

Installation:
-Download or copy the latest code from the repository
-import the file into any system that can read/run python
-place *your* API key into the identified area (line 56)
-Run the code to access the GUI
-Input the location(s) and transport method
-press "calculate"
